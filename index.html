Documentación del API Gateway

NekoExplorer expone una suite robusta y veloz de APIs públicas unificadas listas para integrarse en tus propios sistemas (bots de WhatsApp, Discord, scripts o backends).

Host Base de producción: https://explorador-img-bot.vercel.app

Características clave de la API:

  • Soporte CORS Completo: Todos los endpoints devuelven cabeceras Access-Control-Allow-Origin: *. Esto significa que puedes consultarlos directamente con JS frontend (usando fetch o axios) sin problemas de CORS.
  • Formato JSON Normalizado: Respuestas rápidas codificadas en JSON estándar.
  • Sin API Key ni Autenticación: Libre de límites y keys comerciales.

Gateway de Imágenes: GET /api/v1/image

Obtén una ilustración o foto aleatoria normalizada a partir de múltiples APIs en cascada (nekos.best, nekos.life, waifu.im, nekobot y e621/e926).

Parámetros del Endpoint:

Parámetro Tipo Requerido Descripción
category string Sí Categoría solicitada (ej. neko, waifu, hug, feral, hneko).
provider string No Fuerza el uso de un proveedor específico: nekos.life, nekos.best, waifu.im, nekobot, e621.
nsfw boolean No Indica si se habilita contenido explícito 18+. Valor por defecto false.

NekoExplorer v2.0

Una suite premium avanzada de visualización anime, fur y utilidades de IA.

¿Qué es NekoExplorer v2.0?

Esta aplicación es una interfaz de usuario de una sola página (SPA) moderna, rápida y estilizada diseñada para interactuar con múltiples APIs públicas de arte anime, fur y utilidades de texto. Esta nueva versión incorpora procesamiento distribuido en la nube, almacenamiento estructurado y traducción inteligente.

Características y Capacidades Reales:

  • Integración Multiproveedor: Conexión fluida con Nekos.life, nekos.best, waifu.im, NekoBot.xyz y e621 / e926 (Furry).
  • Almacenamiento Local Robusto (IndexedDB): Utiliza bases de datos locales indexadas en el navegador en lugar de almacenamiento simple de cookies o LocalStorage. Esto permite gestionar tu colección de favoritos de forma ilimitada y guardar un historial de búsquedas recientes cronológico.
  • Búsqueda por Etiquetas Sanitizada: Campo de búsqueda manual exclusivo para e621 que permite consultar etiquetas libres con algoritmos de sanitización estrictos contra inyecciones de código (XSS/HTML).
  • Traducción por IA en Tiempo Real (Gemma): El generador de frases curiosas y existenciales utiliza Serverless Functions de Vercel y la API de Google AI Studio, enviando peticiones en cascada al potente modelo de lenguaje Gemma 4 (gemma-4-31b-it) para traducirlas instantáneamente al español con formato estructurado JSON.
  • Gestión NSFW y Control de Edad: Filtros de edad (Age Gate) interactivos y un switch de seguridad que protege la navegación, conmutando de forma segura entre dominios aptos para todo público (SFW) y de contenido restringido (NSFW).

Sobre la Licencia y los Datos:

Los recursos visuales son propiedad intelectual de los respectivos artistas y proveedores públicos de cada API. NekoExplorer actúa puramente como visor y gestor local de demostración de diseño premium sin fines comerciales. Todas las llamadas a las funciones de traducción y proxy fur están optimizadas mediante la red de borde (Edge/Serverless) de Vercel.
